# 🇸🇳 Senegal Development Intelligence Platform

## 📊 Project Overview

A comprehensive data analytics platform analyzing Senegal's development indicators across multiple dimensions, featuring comparative analysis with West African neighbors and regional breakdowns within Senegal.

## 🎯 Key Features

### 📈 **Analytics Dimensions**
- **Population Demographics**: Official ANSD data (1960-2024)
- **Economic Indicators**: GDP growth, inflation, trade balance, FDI
- **Regional Comparison**: West African neighbors analysis
- **Regional Breakdown**: 11 Senegal regions analysis
- **Trend Analysis**: Forecasting and projections

### 🛠️ **Technical Stack**
- **Data Warehouse**: Snowflake
- **Data Sources**: World Bank API, ANSD Senegal, Enhanced indicators
- **Dashboard**: Streamlit (Mobile-responsive)
- **Visualization**: Plotly
- **Data Processing**: Python, Pandas

### 📊 **Data Coverage**
- **390+ Records** across 6 categories
- **65 Years** of historical data (1960-2024)
- **17 Population** indicators
- **65 Economic** indicators
- **Regional data** for 11 Senegal regions
- **Comparative data** for 7 West African countries

## 🚀 **Key Insights**

### 📈 **Population Trends**
- **5.7x Growth**: 3.34M (1960) → 18.97M (2024)
- **Urbanization**: 23% → 68% (Rapid urban transformation)
- **Growth Rate**: 3.1% annually (2024)
